---

**AIMemLink** is where you write context about yourself — in your own words — and share it with AI through a **private, unguessable link**. Optionally publish a **public profile** with only the basics you choose.

| | Private memory | Public memory |
|---|:---:|:---:|
| **Who can read** | Only people with your secret link | Anyone who knows your username |
| **Best for** | Personal context, work, preferences | Intentionally shareable basics |

No browser extension. No MCP required for reads. Update on the web or via Telegram.
